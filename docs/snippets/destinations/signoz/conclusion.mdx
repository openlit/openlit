### 3. Import the pre-built Dashboard

1. Log into your SigNoz Instance.
1. From the sidebar, choose **Dashboards**.
1. Click **Create new dashboard** button and select **Import JSON** in the drop-down menu.
1. Copy the GenAI Observability dashboard JSON provided [here](https://raw.githubusercontent.com/openlit/openlit/main/assets/signoz-openlit.json).
   1. To get the GPU Monitoring dashboard, copy the dashboard JSON provided [here](https://raw.githubusercontent.com/openlit/openlit/main/assets/signoz-gpu-dashboard.json)
1. Paste the dashboard JSON text directly into the text area.
1. Click **Load Json**.
