OpenLIT is the open-source intelligence layer for teams building with AI. It helps you monitor, debug, and optimize your entire AI stack creating a continuous reliability feedback loop from development to production. OpenLIT is self-hostable and built on OpenTelemetry.

    End-to-end LLM observability and AI agent observability for your entire stack:
    - [OpenTelemetry-native SDKs](/latest/sdk/overview) - Automatic instrumentation for 90+ LLMs, frameworks, vector databases, MCP servers, and GPUs.
    - [Exceptions Monitoring](/latest/openlit/observability/error) - Track and debug errors with full stack traces and span context.
    - [Zero-code Controller](/latest/controller/overview) - eBPF-based discovery and instrumentation across Kubernetes, Docker, and Linux. No SDK, no code change.
    - **Universal compatibility** - Ingest traces from any OpenTelemetry-instrumented tool, including OpenInference and OpenLLMetry.
  </Tab>

  <Tab title="Coding Agents" icon="code">
    First-class observability for AI coding agents including Claude Code, Cursor, and Codex:
    - [One-command setup](/latest/openlit/coding-agents/onboarding) - Install the `openlit` CLI and wire any vendor in under a minute. No SDK, no code changes in your repos.
    - **Per-session visibility** - See cost, lines added and removed, acceptance rate, commits, and PRs per session and per user.
    - **Auto-detected on `/agents`** - Coding agent sessions appear automatically alongside your other agent traffic.
    - **Privacy controls** - Tier-1 secret redaction, personal vs. work classification, and cohort-floor anonymization built in.
  </Tab>

  <Tab title="Otter" icon="message-bot">
    <img src="/images/otter.svg" width="60" className="mb-4" />

    **Otter** is an AI copilot built into OpenLIT. You can query your telemetry, manage platform resources, and generate dashboards through natural language:
    - [Natural language queries](/latest/openlit/chat) - Ask questions about traces, costs, and tokens. Otter converts them to SQL and runs them inline.
    - **Trace and span analysis** - Get structured analysis across performance, reliability, cost, token efficiency, and execution path.
    - **Prompt improvement** - Ask Otter to review and suggest improvements to any Prompt Hub prompt.
    - **Dashboard generation** - Describe the dashboard you want and Otter generates a complete importable layout with working queries.
    - **Resource management** - Create rules, contexts, vault secrets, and custom models through conversation.
  </Tab>
