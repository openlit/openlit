In the OpenLIT, you can add and switch between multiple ClickHouse databases. This feature enables users to maintain separate databases for different environments while using the same OpenLIT to analyze LLM Observability data.

<Frame>
  <img src="/images/multiple-db.png" />
</Frame>

1. **Log in** to the OpenLIT.
2. Go to the **database configuration** page.
3. Click on the **Add new** option.
4. Enter the ClickHouse database configuration details.
5. Click **Save** to confirm.
6. Simply **enable** the desired database to view its data. This selected database will power the dashboards and request page.
