OpenLIT uses OpenTelemetry Auto-Instrumentation to help you monitor applications built on the Model Context Protocol (MCP). This includes tracing MCP server and client JSON-RPC calls, tool execution, and session lifecycle events, giving you AI observability into how your agents discover and call MCP tools.

Auto-instrumentation means you don't have to set up monitoring manually for different LLMs, frameworks, or databases. By simply adding OpenLIT in your application, all the necessary monitoring configurations are automatically set up.

The integration is compatible with the `mcp` Python SDK `>= 0.1.0`.
