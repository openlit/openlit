OpenLIT uses OpenTelemetry Auto-Instrumentation to help you monitor LLM applications built on DigitalOcean's GenAI Platform via the [`pydo`](https://github.com/digitalocean/pydo) SDK. This includes tracking performance, token usage, costs, cache hits, and how users interact with chat completions, embeddings, the Anthropic-style messages API, the OpenAI-style Responses API, and DigitalOcean Agents.

Auto-instrumentation means you don't have to set up monitoring manually for different LLMs, frameworks, or databases. By simply adding OpenLIT in your application, all the necessary monitoring configurations are automatically set up.

The integration is compatible with:
- DigitalOcean `pydo` Python SDK `>= 0.10.0` (sync `pydo.Client` and async `pydo.aio.Client`)
