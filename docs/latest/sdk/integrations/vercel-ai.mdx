OpenLIT uses OpenTelemetry Auto-Instrumentation to help you monitor AI applications built using the Vercel AI SDK. This includes tracking performance, token usage, and how users interact with your application.

Auto-instrumentation means you don't have to set up monitoring manually for different providers or models. By simply adding OpenLIT in your application, all the necessary monitoring configurations are automatically set up.

The integration is compatible with  
- Vercel AI SDK (`ai` package)
