OpenLIT uses OpenTelemetry Auto-Instrumentation to help you monitor AI agents built with Microsoft Agent Framework. This includes tracing agent construction, agent execution, tool invocation, and workflow runs, giving you AI observability into multi-agent and multi-step workflows.

Auto-instrumentation means you don't have to set up monitoring manually for different LLMs, frameworks, or databases. By simply adding OpenLIT in your application, all the necessary monitoring configurations are automatically set up. OpenLIT disables Agent Framework's built-in telemetry layer and replaces it with OTel GenAI semantic-convention-compliant spans, so you don't get duplicate traces.

The integration is compatible with `agent-framework >= 1.0.0rc1`.
